# FlightFits

A runnable Expo Router starter for an AI-assisted travel packing app.

## Included

- Onboarding
- Login and signup mock screens
- Bottom tabs
- Home, Trips, Closet, and Profile screens
- Create Trip form
- Rule-based packing list generator
- Interactive checklist and progress bar
- Placeholder files for Supabase, weather, and AI integrations
- Shared components, constants, types, hooks, services, and utilities

## Install

From Terminal:

```bash
cd ~/flightfits
npm install
npx expo start --clear
```

## Replace your current files safely

1. Rename your existing project folder or commit it to Git.
2. Copy the contents of this starter into the project root.
3. Keep your own image assets if you already added any.
4. Run `npm install`.
5. Run `npx expo start --clear`.

## Current limitations

Authentication, database saving, weather, closet uploads, and AI are mock placeholders. The starter runs without API keys so you can build the interface and packing workflow first.
